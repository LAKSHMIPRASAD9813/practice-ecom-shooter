🌐 HTML Basics (Hyper Text Markup Language):


Hyper = Reference or Link 📎


Text = Data 📝


Markup = Tags 🏷️


HTML uses tags to structure content, and every open tag must have a closing tag. Example:


html.../html



🧱 Basic HTML Structure:


Every HTML document follows a standard structure to ensure proper functionality. Here’s a simple layout:


!DOCTYPE html
html lang="en"
  head
    meta charset="UTF-8"
    meta name="viewport" content="width=device-width, initial-scale=1.0"
    meta http-equiv="X-UA-Compatible" content="IE=edge"
    titleMy Webpage/title
  /head
  body
    h1Welcome to My Website!/h1
    pThis is my journey into web development!/p
  /body
/html

🔍 Understanding Metadata:


Metadata provides information about the HTML document but doesn't appear on the page itself. It helps with search engine optimization (SEO), responsive design, and browser compatibility.


meta tags in the head section define document properties like character encoding, viewport settings, and SEO keywords for better visibility.

🚀 #DAY7 of My Frontend Development Journey 🚀


Today’s Focus: Making websites dynamic with multimedia elements! Adding images, audio, and video can transform a webpage, making it more interactive and engaging for users. Here’s what I dived into:



🖼️ Working with Images


img Syntax: img src="image.jpg" alt="description" width="300" height="200"


Key Attributes:


    
    
  • src: Link to the image file.
    • 
    
  • alt: Text for accessibility and SEO boosts.
    • 
    
  • width & height: Adjusts image dimensions directly in the HTML.
    • 




🎶 Adding Audio Clips


audio Syntax: audio controlsYour browser does not support the audio tag.


Key Attributes:


    
    
  • controls: Adds play, pause, and volume controls.
    • 
    
  • loop, autoplay, muted: Custom playback options for a seamless user experience.
    • 




📹 Embedding Videos


video Syntax: video width="320" height="240" controlsYour browser does not support the video tag.


Key Attributes:


    
    
  • width & height: Set video dimensions.
    • 
    
  • controls: Adds playback options for a better UX.
    • 
    
  • autoplay, muted, loop: Manage playback settings to fit the context.
    • 




Why Multimedia? Enhancing the visual and interactive elements of a website improves user engagement and keeps them coming back for more!
